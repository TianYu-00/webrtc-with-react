# WebRTC with React
A real-time communication platform with video, audio, and screen sharing, built with Node.js, Express, Socket.IO, WebRTC, and a React frontend styled with Tailwind CSS.

- Socket.IO

``` bash
npm install
npm start
```

- React (Vite)
- Tailwind
- WebRTC
- Socket.IO-Client
- React-Icons

``` bash
cd frontend
npm install
npm run dev
```
